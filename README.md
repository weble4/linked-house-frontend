# Linked-House
*Front Repository*

### 프로젝트 개요
Weble4 팀의 프로젝트 linked-house front는 airbnb 모델을 벤치마킹하여, 숙박 공유 사이트를 만드는 것을 목표로 합니다.
프론트에서는 백엔드와의 API 통신을 통해 데이터를 가지고와 화면을 구성합니다..
